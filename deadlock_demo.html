🛡️ How to Prevent This Deadlock

The deadlock occurred because P1 and P2 requested resources in opposite order. Here's how to prevent it:

1. Resource Ordering

Always request resources in the same order (e.g., Scanner first, then Printer). This breaks the circular wait condition.

2. Preemption

Allow the OS to forcibly take resources from processes. If P1 is waiting, P2's Scanner could be preempted.

3. Request All at Once

Require processes to request all needed resources atomically. If one isn't available, none are allocated.

4. Banker's Algorithm

Use deadlock avoidance by checking if allocation would lead to an unsafe state before granting resources.
