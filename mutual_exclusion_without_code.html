🧠 Key Concepts

Race Condition: When two threads access shared data concurrently and at least one writes, the outcome depends on timing.

Critical Section: Code that accesses shared data and must not be executed by more than one thread at a time.

Mutual Exclusion: Ensuring only one thread is in the critical section at any time.
